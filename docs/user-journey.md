# User Journey — The Nostalgic Fan

## Scenario

A user wants to relive the feeling of a favorite live performance and keep exploring similar moments without losing momentum.

| Stage | User Action | Thought / Feeling | Pain Point | Opportunity |
|---|---|---|---|---|
| Entry | Opens Encore Atlas | “I want to revisit that feeling.” | Doesn’t know exact next search yet | Make entry feel inviting and memory-driven |
| Search | Searches for an artist or performance | Hopeful, curious | Search is still required to begin | Support returning-user recommendations over time |
| Watch | Plays a live video | Engaged, nostalgic | Session can end after one video | Keep immersion going after playback |
| Discover | Sees recommended videos | “Yes, this is the same vibe.” | Recommendations may not always feel relevant early on | Improve through tags + user behavior |
| Continue | Clicks another video | Pulled deeper into the experience | Too few results could stall exploration | Expand recommendations when data/API limits improve |
| Return | Comes back later | Wants faster re-entry next time | Current landing is search-first | Personalize homepage for returning users |
